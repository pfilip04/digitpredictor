Welcome to the Digit Predictor Web App!

This web application uses a convolutional neural network (CNN) to recognize hand-drawn digits written directly on a digital canvas. It’s built with Django and powered by a custom-trained machine learning model.

How to Use:

  1. Go to the Digit Predictor page.
  2. Write a digit (0-9) on the canvas using your mouse or touchscreen.
  3. Click the "Predict" button.

The app will analyze your drawing and return the top 3 predictions based on the model’s confidence.

Note on Accuracy:

  • The confidence scores are based on the softmax function, which may not always reflect true certainty.

For best results:

  • Keep the digit natural — avoid stretching, tilting, or adding extra noise.
  • Try not to overexaggerate the digit's features; subtle and clean works best.
  • The larger the digit the better!

Start with drawing!
